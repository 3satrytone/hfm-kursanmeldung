Kursanmeldung (Course Registration)

This is a minimal TYPO3 v13 extension scaffold.

Vendor: HFM
Extension key: kursanmeldung
Composer package: hfm/kursanmeldung

How to use
- Ensure the project is installed and TYPO3 v13 is set up.
- Run `composer dump-autoload` in the project root so TYPO3 detects the extension from the packages/ path repository.
- In the TYPO3 backend, go to Extensions and install the extension "Kursanmeldung".

Development
- PHP: >= 8.2
- TYPO3: ^13.4
- Namespace: Hfm\\Kursanmeldung

Structure
- Classes/: PHP code (PSR-4)
- Configuration/: future configuration files
- Resources/: assets and language files
